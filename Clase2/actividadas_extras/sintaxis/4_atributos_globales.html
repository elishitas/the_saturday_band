Atributos Globales

Globales significa que se puede aplicar a cualquier etiqueta de HTML

Agregan ciertos datos a las etiquetas

Recorda que HTML es un lenguaje de marcardo y que todo lo que marcamos tiene que ver con el contenido sematico de la página

Atributo Title

No hay que confundirlo con la etiqueta title que se encuentra en el head

Ejemplo

Este párrafo contiene un link que me lleva a Google

¿Cuál es el funcionamiento? Va decir donde me va a llevar ese link

El cartelito que aparece es el valor que tiene el atributo title

Atributo ID

Se tiene que evitar usar caracteres extraños, se puede utilizar:letras, números, guión medio / bajo

Podes aplicar todos los id que quieras pero debe ser único en la página, no podes repetir nombres de ID

¿Para qué sirve aplicar un identificador a un elemento?Todavía no vimos CSS o Js, cuando lo empiecen a utilizar el ID es fundamental

Class o clase en español

Es un atributo parecido al ID

La diferencia es que podemos aplicar a varios elementos la misma clase

El nombre de la clase lo decidis vos

Atributo lang

Lang viene de lenguage

Indica en que lenguaje -español, inglés, alemán, etc- está escrito el contenido de cualquier etiqueta de HTML
